## 🇬🇧 English

Thank you for your interest in contributing to **Andoriña**! This is an independent, community-driven project aiming to turn Hermes into an autonomous WhatsApp manager. Whether you're fixing bugs, improving documentation, or proposing new features, your help is incredibly valuable.

### How to Contribute

#### 1. Reporting Bugs
Found a bug? We want to know!
- Check the [Issues](https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes/issues) tab to see if it has already been reported.
- If not, open a new issue using our **Bug Report template**.
- Provide as much context as possible (OS, Python version, Hermes version, logs from the terminal).

#### 2. Suggesting Enhancements
Have an idea to make Andoriña even smarter?
- Open an issue using the **Feature Request template**.
- Explain *why* this feature would be useful and how it aligns with our philosophy of privacy, anti-spam, and automation.

#### 3. Submitting Pull Requests
Want to write some code? Awesome!
1. **Fork the repository** and create your branch from `main`.
2. Keep your changes focused. If you're fixing a bug and adding a feature, please do it in two separate PRs.
3. **Follow the vibe:** Ensure your code is clean, well-commented, and respects the existing architecture (Multi-agent isolation, Security Guard, etc.).
4. Test your changes locally to ensure they don't break existing functionality.
5. Create a Pull Request with a clear description of what you've done.

### Project Philosophy
- **Anti-Spam:** We strictly prohibit features designed for mass-marketing or spam.
- **Privacy First:** All data must be processed locally. No external APIs (unless explicitly opted-in by the user, like Google Contacts).
- **Resilience:** Code should gracefully handle bridge disconnects or missing permissions.

## 🇪🇸 Español

¡Gracias por tu interés en contribuir a **Andoriña**! Este es un proyecto independiente impulsado por la comunidad con el objetivo de convertir a Hermes en un gestor autónomo de WhatsApp. Ya sea arreglando bugs, mejorando la documentación o proponiendo nuevas funciones, tu ayuda es increíblemente valiosa.

### Cómo Contribuir

#### 1. Reportar Errores (Bugs)
¿Has encontrado un fallo? ¡Queremos saberlo!
- Revisa la pestaña de [Issues](https://github.com/AndorinaAI/Andorina-WhatsApp-Agent-for-Hermes/issues) para ver si ya ha sido reportado.
- Si no, abre una nueva issue usando nuestra **plantilla de Bug Report**.
- Proporciona todo el contexto posible (Sistema Operativo, versión de Python, versión de Hermes, logs del terminal).

#### 2. Sugerir Mejoras
¿Tienes una idea para hacer Andoriña aún más inteligente?
- Abre una issue usando la **plantilla de Feature Request**.
- Explica *por qué* esta función sería útil y cómo se alinea con nuestra filosofía de privacidad, anti-spam y automatización.

#### 3. Enviar Pull Requests
¿Quieres escribir código? ¡Genial!
1. Haz un **Fork del repositorio** y crea tu rama a partir de `main`.
2. Mantén tus cambios enfocados. Si vas a arreglar un bug y añadir una función, por favor hazlo en dos PRs separados.
3. **Sigue el estilo:** Asegúrate de que tu código esté limpio, bien comentado y respete la arquitectura existente (Aislamiento Multi-agente, Guardia de Seguridad, etc.).
4. Prueba tus cambios localmente para asegurar que no rompen funciones existentes.
5. Crea un Pull Request con una descripción clara de lo que has hecho.

### Filosofía del Proyecto
- **Anti-Spam:** Prohibimos estrictamente funciones diseñadas para marketing masivo o spam.
- **Privacidad Primero:** Todos los datos deben procesarse localmente. Sin APIs externas (a menos que el usuario lo habilite explícitamente, como Google Contacts).
- **Resiliencia:** El código debe manejar con gracia las desconexiones del puente o la falta de permisos.

## 🇬🇧 English

### Supported Versions

Andoriña is an actively developed project. We highly recommend always using the **latest stable version** available on the `main` branch to ensure you have the latest security patches, anti-ban mechanisms, and privacy features.

| Version | Supported | Notes |
| ------- | :---: | --- |
| **v1.0.x** (Latest) | ✅ | Receives all security, anti-ban, and feature updates. |
| **< v1.0.0** | ❌ | Deprecated. Please upgrade immediately. |

### Reporting a Vulnerability

Your security and privacy are our absolute priority. Given the nature of this project (WhatsApp integration, local AI agents, and personal data handling), we take all vulnerability reports extremely seriously.

If you discover a security vulnerability or a privacy flaw (such as path traversal, prompt injection bypasses, or data leaks), please **DO NOT** open a public issue.

Instead, please send an email directly to the project maintainer:
**Email:** `jorgegformoso@gmail.com`

**What to expect:**
- We will acknowledge receipt of your vulnerability report within 48 hours.
- We will provide an estimated timeline for the fix and the public disclosure.
- We will credit you in the release notes if you wish.

### Security Philosophy
Andoriña is built with a **Privacy-First, Local-Only** approach. We rely on the community to help us maintain a robust firewall (`guard.py`) against prompt injections and a resilient architecture against unauthorized access. Thank you for keeping Andoriña safe!

## 🇪🇸 Español

### Versiones Soportadas

Andoriña es un proyecto en desarrollo activo. Recomendamos encarecidamente utilizar siempre la **última versión estable** disponible en la rama `main` para asegurar que dispones de los últimos parches de seguridad, mecanismos anti-baneo y funciones de privacidad.

| Versión | Soportada | Notas |
| ------- | :---: | --- |
| **v1.0.x** (Última) | ✅ | Recibe todas las actualizaciones de seguridad, anti-baneo y funciones. |
| **< v1.0.0** | ❌ | Obsoleta. Por favor, actualiza inmediatamente. |

### Reportar una Vulnerabilidad

Tu seguridad y privacidad son nuestra prioridad absoluta. Dada la naturaleza de este proyecto (integración con WhatsApp, agentes locales de IA y manejo de datos personales), nos tomamos todos los informes de vulnerabilidades muy en serio.

Si descubres una vulnerabilidad de seguridad o un fallo de privacidad (como un salto de directorio, evasiones de inyección de prompts o fugas de datos), por favor **NO** abras una issue pública.

En su lugar, envía un correo electrónico directamente al mantenedor del proyecto:
**Correo electrónico:** `jorgegformoso@gmail.com`

**Qué esperar:**
- Confirmaremos la recepción de tu informe de vulnerabilidad en un plazo de 48 horas.
- Proporcionaremos un plazo estimado para la corrección y la divulgación pública.
- Te daremos crédito en las notas de la versión si así lo deseas.

### Filosofía de Seguridad
Andoriña está construida con un enfoque de **Privacidad Primero, 100% Local**. Confiamos en la comunidad para ayudarnos a mantener un cortafuegos robusto (`guard.py`) contra inyecciones de prompts y una arquitectura resiliente contra accesos no autorizados. ¡Gracias por mantener a Andoriña segura!

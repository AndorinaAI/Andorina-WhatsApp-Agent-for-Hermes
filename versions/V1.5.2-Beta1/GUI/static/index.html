Configurar Contacto / Grupo Específico

Selecciona un contacto o grupo para configurar sus permisos, rol y personalidad.

Si se define, el bot solo responderá si el mensaje empieza EXACTAMENTE por esta palabra (Ideal para simular múltiples personajes o menciones personalizadas como '@Zack').

Si defines etiquetas aquí, cuando este usuario pida al bot buscar en la agenda, el bot solo podrá ver los contactos que tengan esa etiqueta (aislando el resto de tu agenda).

Permite que este usuario inicie sesión en el panel (su rol define qué ve).

Si dejas esto vacío, el bot le atenderá en todos los grupos. Si añades el ID de un grupo aquí, el bot lo ignorará en privado y en otros grupos, limitándole solo al sitio indicado.

Límites de Memoria Automática (Core)

Límite de Usuario (USER.md): Expediente individual. Lo que la IA aprende de la persona específica con la que está hablando (ej: "Se llama Juan, le gusta el color azul, compró un portátil").

Límite Global (MEMORY.md): Conocimiento general. Lo que la IA aprende sobre sí misma o su entorno global (ej: "Nuestra tienda cierra a las 20:00", "Hoy es festivo"). Es un conocimiento compartido para todos.

💡 Aumentar estos límites (ej. a 4000) permite a la IA recordar expedientes masivos sin borrar datos viejos, pero consume más tokens de la API.

Valores de Referencia:
  • 1000 - 1500 (Recomendado): Nombres, gustos y detalles simples.
  • 2000 - 3000 (CRM Básico): Historial de pedidos, preferencias de clientes, estado de negociaciones.
  • 4000 - 5000 (Intensivo): Fichas médicas, historiales técnicos de soporte, actas detalladas.
  • 6000+ (Experto): Para asistentes personales hiper-personalizados. (⚠️ Alto consumo).
Valores de Referencia:
  • 2000 - 2500 (Recomendado): Estado del bot, reglas básicas, fechas y horas.
  • 3000 - 4500 (Empresa): Horarios, normativas completas, avisos globales de la tienda.
  • 5000+ (Corporativo): Manuales operativos enteros aprendidos orgánicamente. (⚠️ Alto consumo).

Asistente de Instalación

                
Instala y configura Andoriña en un entorno local o
                    de servidor sin usar la consola.

            


            
            

                
Sin acceso directo instalado

                
Para abrir el panel en el futuro sin usar la terminal,
                    instala el acceso directo en el paso correspondiente. Mientras tanto, para abrir el panel:

                
cd /ruta/actual/de/la/carpeta
python3 GUI/server.py
→ Luego abre http://localhost:8888

                
            


            
            

                
✅ Panel Instalado Permanentemente

                
El panel y el asistente se han copiado con éxito a tu
                    carpeta oculta de Hermes. Ya puedes cerrar esta ventana y borrar la carpeta de descargas
                        original de Andoriña. Para volver a abrir este panel en el futuro, utiliza el acceso directo
                    que se ha creado en el menú de aplicaciones de tu sistema.

                    Ver Instrucciones de Instalación Manual (Fallback)
                    

                        
Si algún paso falla desde la
                            interfaz gráfica, puedes ejecutar estos comandos en tu terminal.


                        
Dependencias

                        
pip install --user requests pyyaml python-dotenv google-auth google-auth-oauthlib google-api-python-client
# Si el anterior falla en Ubuntu/Debian moderno:
pip install --user --break-system-packages requests pyyaml python-dotenv google-auth google-auth-oauthlib google-api-python-client


                        
Instalación interactiva desde terminal

                        
cd /ruta/de/la/carpeta/descargada
python3 setup.py
# O con el script de shell:
bash install.sh


                        
Vincular Google Contacts a mano

                        
cd /ruta/de/la/carpeta
python3 scripts/auth.py


                        
Desplegar solo los scripts a mano

                        
cp -r scripts/ ~/.hermes/skills/andorina/scripts/
cp SKILL.md ~/.hermes/skills/andorina/SKILL.md
